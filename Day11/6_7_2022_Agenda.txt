Steps for implementing ORM
1.Create a blank solution using Visual studio.net
2.Create empty asp.net framework  with MVC folder inside application
3.Download Entityframework and Mysql.Data.Entityframework packages using Nugget Package manager.
4.Add POCO class Product in Models folder
5.Map Product class with Products table using [Table] attribute.
6.Add new folder DAL and create new DbOrmContext class .
7.Dervice DbOrmContext Class from DBContext class.
8.Add data member DbSet <Product> Products {get;set} property to DbOrmContext class.
9.Add Connectionstring entry in web.config file
10.Register Connectionstring with base class constructor in DbOrmContext class.
